## お弁当注文システム

### プロダクトバックログ（MVP）
[x] 今週注文できるお弁当リストが表示できる
[x] お弁当が注文できる（注文した結果はメールでは送られない）
[x] お弁当屋さんに明日の注文リストをLINEで送る
[] 来週のお弁当を更新できる
[　] お弁当メニュー表示を最新のものに切り替える
[] 毎日6:30に明日の注文リストをLINEで送る
[] 注文した結果をメールで送る

### プロダクトバックログ（オプション）
[] お弁当を注文した結果をメールで送る
[] 6:30以降に明日の注文をした場合、注文できませんという
[] 注文のキャンセルができる

注文フォームのvalidation
* お弁当注文
  [x] 注文個数は数値でないといけない。（プルダウンメニューの方がいいかも）
  [x] 氏名が空欄ではいけない
  [x] 連絡先はemailアドレスでないといけない。
  [x] 受け取り日は明日以降でないといけない。
  [x] 受け取り日は平日でないといけない。
  [x] 注文した時間が６：３０以降の時は明日の注文はできない。
